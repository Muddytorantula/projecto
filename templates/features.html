All-in-one project management

Running different systems is tough. Getting people to use them is tougher. Projecto is designed with all the features you need in a single system. Get it all for free!

Need a new feature? Just add it yourself and don't forget to send a pull request!

One Feed for Everything

With the project feed, each member of the team can post updates, ask questions, or hold discussions. A single destination allows everyone to easily track what everyone else is doing.

If this were an Apple product, this would definitely be a beautiful feed.

Collaborate Using Todo Lists

Each project gets its very own multiplayer todo list. Work efficiently with your team by assigning and tagging individual items.

Discuss any issue using its comments section.

Share Files Effortlessly

Shared file storage is essential to any team. Projecto offers each project its very own shared folder, perfect for engineering schematics, budget spreadsheets, or funny cat pictures. Keep your files organized in subfolders.

Just like everything else in Projecto, each file has its own comments section, enabling lively debates.

Manage Your Team Like a Boss

Simply add your team members' email addresses. Projecto uses Mozilla Persona for an easy and secure sign in experience.

Projecto works with all email providers, including Gmail and Yahoo. For additional security, you could even implement your own authenticator for your domain.

Open source

Projecto's source code can be found on github. It is licensed under Apache 2. This means you are free to host, modify, and distribute your own version. Just remember, sending a pull request would get you srs nerdcreds.
